# 🛒 Labrostone - Full Stack E-Commerce Platform

Labrostone is a feature-rich, fully functional E-Commerce web application built using the **MERN Stack** (MongoDB, Express.js, React, Node.js). Inspired by platforms like Flipkart, it offers a seamless shopping experience for users and a robust management system for admins.

---

## 🚀 Key Features

### 👤 User End Features
* **Authentication & Authorization:** Secure Login/Signup with JWT (JSON Web Tokens) and Google OAuth.
* **Product Browsing:** Advanced filtering, sorting, and search functionality for products.
* **Shopping Cart & Wishlist:** Fully interactive cart and wishlist with real-time updates.
* **Address Management:** Add, edit, or delete multiple shipping addresses.
* **Secure Checkout & Payments:** Integrated payment gateway (like Razorpay/Stripe) for smooth transactions.
* **Order History:** Track current orders and view complete past order history with invoice downloads.

### 👑 Admin Panel Features
* **Dashboard:** Graphical analytics of total sales, users, and top-selling products.
* **Product Management:** Full CRUD (Create, Read, Update, Delete) operations for products, categories, and inventory.
* **Order Management:** Update order status (Processing, Shipped, Delivered) and manage returns.
* **User Management:** View and manage registered users and admin access.

---

## 🛠️ Tech Stack Used

* **Frontend:** React.js, Redux Toolkit (State Management), Tailwind CSS / Bootstrap, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Mongoose ODM)
* **Authentication:** JWT, bcryptjs
* **File Uploads:** Cloudinary (for product images)
